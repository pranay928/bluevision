BlueVision
BlueVision is a simple and elegant multi-page website template designed for modern businesses, portfolios, or service providers. It includes responsive layouts, clean design elements, and ready-to-use HTML pages to help you quickly launch your project.

📂 Project Structure
index.html — Home page

portfolio-details.html — Portfolio/project detail page

service-details.html — Service detail page

starter-page.html — Starter template page

deletedsection.html — Backup or removed sections

forms/ — Placeholder for form-related files

assets/ — Contains CSS, JS, images, and fonts

✨ Features
Clean and responsive design

Modular and reusable components

Lightweight and fast

Easy to customize

Organized folder structure

🛠️ Technologies Used
HTML5

CSS3

JavaScript

PHP (minor usage for form handling)
